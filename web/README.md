# Apex Sender - Web Interface

## 🌐 كيفية الاستخدام

### 1. تثبيت المتطلبات
```bash
pip install Flask
```

### 2. تشغيل التطبيق
- شغّل Apex Sender
- اذهب لتاب "Web"
- اضغط "تشغيل الخادم"

### 3. الوصول من الموبايل

#### الطريقة 1: QR Code (الأسهل) ⭐
1. امسح رمز QR الظاهر في التطبيق
2. اضغط على الرابط
3. تفتح الصفحة!

#### الطريقة 2: كتابة الرابط
1. افتح المتصفح على الموبايل
2. اكتب: `http://192.168.1.X:8080`
3. (استبدل X برقم IP الظاهر)

### 4. إرسال ملف
1. اضغط "اختر الملف"
2. اختر الملف من الموبايل
3. أدخل IP المستقبل
4. اضغط "إرسال"

## 📱 حفظ على الشاشة الرئيسية

### Android:
1. افتح الصفحة في Chrome
2. القائمة (⋮) → "Add to Home screen"
3. يصبح مثل التطبيق!

### iPhone:
1. افتح الصفحة في Safari
2. زر المشاركة (□↑) → "Add to Home Screen"
3. يصبح مثل التطبيق!

## ⚙️ المتطلبات

- نفس الشبكة (WiFi) للكمبيوتر والموبايل
- Apex Sender يعمل على الكمبيوتر
- متصفح حديث على الموبايل

## 🐛 استكشاف الأخطاء

### لا يفتح الرابط؟
- تأكد أن الكمبيوتر والموبايل على نفس WiFi
- تأكد أن Firewall لا يمنع Port 8080

### الإرسال يفشل؟
- تأكد أن IP المستقبل صحيح
- تأكد أن المستقبل يعمل على Port 5001
