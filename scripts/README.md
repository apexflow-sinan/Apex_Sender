# Scripts Folder

Helper scripts for development and maintenance.

## Files

- **check_dependencies.py** - Check if all required packages are installed
- **download_sounds.py** - Guide for adding custom sound files
- **generate_icon.py** - Old icon generator (deprecated)
- **generate_icons.py** - Generate multi-size icons from main icon
- **run_debug_start.bat** - Run application with debug output
- **run_no_console.bat** - Run application without console window
- **test_app.py** - Test application modules

## Usage

### Check Dependencies
```bash
python scripts/check_dependencies.py
```

### Generate Icons
```bash
python scripts/generate_icons.py
```

### Run Without Console
```bash
scripts/run_no_console.bat
```

### Debug Mode
```bash
scripts/run_debug_start.bat
```
